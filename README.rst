Sliding Window over Matrix, and Vector
======================================

Classes
-------

- PowerMatrix
- PowerVector


Example
-------

Given the following vector:

[1, 2, 3, 4, 5]

A sliding window can have a width of 3.

Sliding window can be located at position 0, 1 or 2

At position 0

[**1**, **2**, **3**, 4, 5]

At position 1

[1, **2**, **3**, **4**, 5]

At position 2

[1, 2, **3**, **4**, **5**]
